    Reflections

    I belevie this project was well orginized with scripts being made simple and modular.

  • I struggled with the KISS princible as i over designed the character and animations for the size of the project I also kept adding more features to the game that made it more complex than it needed to be like the powerups and the cabinet buttons working. A downfall of this is the lack of feedback to the user when events happen
  • A problem i did not expect was getting the ball to deflect correctly, my original idea was to use magnitude to invert the direction but i coudnot get that to work, so i ended up inverting the balls X or Y depending on what was hit
